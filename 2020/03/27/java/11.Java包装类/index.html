包装类

包装类是为了解决基本数据类型没有属性和方法的问题.包装类没有子类.

  • Byte
  • Short
  • Integer
  • Long
  • Float
  • Double
  • Character
  • Boolean

字符串和基本类型之间的转换

注意

  • 包装类作为成员属性值时，默认值为null

  • 包装类自动装箱时会有一个对象缓存池/常量池(-128~127),范围内外调用方法不同,float/double除外

  • 和字面量比较时会自动拆箱

    Integer i = 10;
Integer j = new Integer(10);
i == 10; // true
i == j; // false
